🍳 Cooking Buddy
Smart Recipe Generator Based on Your Pantry
A lightweight web app that suggests recipes using only the ingredients you currently have at home. It also shows missing ingredients and their estimated additional cost.

✨ Features
* Add, edit, and delete pantry ingredients
* Generate recipes based on available ingredients
* View recipe details with "You Have" vs "Missing" ingredients
* Show estimated cost for missing ingredients (in PHP)
* Save favorite recipes locally
* Fully responsive design (desktop & mobile friendly)
* All data saved in browser LocalStorage – no backend required

🛠 Tech Stack

* Frontend: HTML5, CSS3, Vanilla JavaScript
* Recipe API: TheMealDB (free public API)
* Storage: Browser LocalStorage
*  Design Tool: Figma
*  Deployment: Netlify / Vercel

🚀 How It Works
1. Add the ingredients you have in your pantry
2. Click "Generate Recipes"
3. Browse matching recipe suggestions
4. Open any recipe to see full instructions, ingredient breakdown, and estimated cost for missing items
